PRIMEROS PASOS

  1. ¿Qué es el vino?

    El vino es una bebida alcohólica elaborada con uvas fermentadas. Técnicamente, el vino se puede hacer con cualquier fruta, pero sólo puede denominarse "vino" en la etiqueta si hace con uvas para vino.

  2. Uvas para vino

    Las uvas de vino son diferentes a las uvas de mesa (las que encontramos en los mercados). Son más pequeñas, más dulces, contienen semillas y tienen una piel más gruesa. Estos rasgos han demostrado ser mejores para hacer vino.

  3. La vid

    La vid es la planta que produce las uvas. Existen muchas especies de vid, de las cuales, la Vitis Vinifera es la más importante para la elaboración de vinos.

  4. Vitis Vinifera

    Hay miles de variedades de vino diferentes y la mayoría son cultivares de una sola especie de vid: Vitis Vinifera.

  5. Característica principal

    Las vides son plantas leñosas que duran mucho tiempo y producen una cosecha una vez al año. Los diferentes climas donde crecen las uvas afectan el sabor dulce (o ácido) del vino resultante.

  6. Tipos de vinos

      Varietal: es un vino elaborado principalmente con una sola variedad de uva.
      Blend: se elabora mezclando varios vinos varietales.
      Co-fermentado: se elabora con diferentes variedades que se cosechan y vinifican juntas.

ENTENDIENDO LA ETIQUETA

  1. Variedad de uva: Cepa utilizada para elaborar el vino.
  2. Añada: Año en el que se cosechó la uva.
  3. Crianza: Referencia para saber cuanto tiempo puede guardrase un vino antes de ser consumido.
  4. Región: Orgien de procedencia de las uvas.
  5. Graduación alcoholica: Indica el grado alcohólico del contenido.
